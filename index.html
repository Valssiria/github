O que é GIT e github?

Git é um programa para fazer versões dos seus códigos. Ele detecta alteração do seu código vamos pegar um repositório (Uma pasta) Seus arquivos. aí você pega a sua pasta e fala que é seu repositório (git). Agente consegue ver todos as alterações feita na pasta. Conseguimos comparar como era nosso código antes e para o como agora ta feito por (comitês). Commite é uma foto daquele exato momento daquele momento do código.
